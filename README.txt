2. CLASE PRINCIPAL PARA EJECUTAR LA SIMULACIÓN
----------------------------------------------
Clase a ejecutar:

    EVDemo.java

(específicamente el método "main()")

Descripción:
    Esta clase contiene el punto de entrada del programa y permite iniciar
    la simulación completa del proyecto a través del método "main()" que 
    se encarga de ejecutar todo, según el nivel de prueba que hayas
    establecido en la misma clase (SIMPLE, MEDIUM o ADVANCED).
